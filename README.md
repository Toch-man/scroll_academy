Scroll Academy is a decentralized learning platform built on blockchain technology. It allows users to access educational content, track progress, and manage credentials securely and transparently. The platform leverages smart contracts for authentication, course enrollment, and certificate issuance, providing a Web3-powered learning experience.

Features

User Authentication: Secure login and role-based access using Ethereum wallets.

Course Management: Instructors can create, update, and manage courses.

Progress Tracking: Learners can track their learning journey.

Certificate Issuance: Blockchain-backed certificates for course completion.

Frontend: Built with React and TypeScript for a fast and responsive user experience.

Blockchain Integration: Uses Hardhat for smart contract development and WAGMI for frontend interaction.

Tech Stack

Frontend: React, TypeScript, TailwindCSS

Blockchain: Solidity (Hardhat), WAGMI, Ethers.js

Deployment: Scroll Sepolia Testnet

State Management: Context API / React hooks

Getting Started

Clone the repository:

git clone https://github.com/yourusername/scroll-academy.git


Install dependencies:

yarn install


Deploy smart contracts:

yarn deploy --network scrollSepolia


Start the frontend:

yarn dev
